# Chirply

Hear a bird. Know its name.

Tap **Listen**, capture a call, and see which bird it is — with soft sound-wave animations and a living bird result.

## Name options

| Name | Why it works |
|------|----------------|
| **Chirply** | Friendly, memorable, clearly about bird sound |
| **Trill** | Short, musical, one syllable |
| **Wingcall** | Literal: bird call identification |
| **Songmark** | “Bookmark” a song you heard outside |
| **Avia** | Clean / premium (from *Aves*) |
| **Dawnchorus** | Poetic, nature-first |
| **Callnote** | Practical field-guide vibe |

**Working name:** Chirply (easy to change).

## Core loop

1. Open app → big **Listen** button  
2. Record ambient bird sound  
3. Match against bird-call library  
4. Show bird + confidence + animated waveform / bird motion  
5. Save to library / learn more

## Design direction

- Dawn forest atmosphere (sage, soft gold, mist)  
- One hero action: Listen  
- Result: bird illustration + name + match % + sound animation  
- No cluttered dashboard in the first screen
